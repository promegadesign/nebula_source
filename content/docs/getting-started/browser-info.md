## Supported Browsers

The promega.com website supports the following browsers:
- Chrome
- Firefox
- Safari
- Edge


## Display Breakpoints

Promega.com has six breakpoints, named for science stuff.

| Breakpoint Name | Range       |
| :-------------- | :---------: |
| Gene            | 0-400px     |
| Chromosome      | 401-480px   |
| Nucleus         | 481-600px   |
| Cell            | 601-768px   |
| Colony          | 769-1024px  |
| Organism        | 1025-1280px |
